# AI Memory Visualizer + Valgrind Tracer

Base del proyecto: [hbtn-edu/ai_memory_visualizer_valgrind_tracer](https://github.com/hbtn-edu/ai_memory_visualizer_valgrind_tracer).

En este repositorio se sube solo el contenido del directorio `analysis/` según lo indicado en cada tarea.

## Build (programas del repo oficial)

```bash
cd programs
make
```

## Run

```bash
./stack_example
./heap_example
./aliasing_example
./crash_example
```

Nota: `crash_example` está pensado para provocar un fallo de segmentación (NULL dereference).
